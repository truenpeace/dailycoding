텍스트를 한 줄로 표시하는 태그

strong 태그, b 태그 - 굵게 표시하기

이것은 스트롱 태그 입니다
이것은 b 태그입니다

em 태그, i 태그 - 이탤릭체로 표시하기

이것은 em 태그 입니다
이것은 i 태그 입니다

q 태그 - 인용 내용 표시하기

이것은 q 태그 입니다

mark 태그 - 형광펜 효과 내기

이것은 mark 태그 입니다

span 태그 - 줄바꿈 없이 영역 묶기

이것은 span 태그 입니다

ruby 태그 - 동아시아 글자 표시하기

이것은 ruby태그주석처리 입니다

기타 텍스트 관련 태그

abbr 태그 - 약자 표시, title 속성을 함께 사용

이것은 abbr 태그 입니다

cite 태그 - 웹 문서나 포스트에서 참고 내용 표시

이것은 cite 태그입니다
웹 문서나 포스트 내용

code 태그 - 컴퓨터 인식을 위한 소스 코드

이것은 code 태그 입니다

kbd 태그 - 키보드 입력, 음성 명령 같은 사용자 입력 내용

이것은 kbd 태그 입니다

small 태그 - 부가 정보처럼 작게 표시해도 되는 태그

이것은 small 태그 입니다

sub 태그 - 아래 첨자

이것은 sub 태그 입니다

sup 태그 - 위 첨자

이것은 sup 태그 입니다

s 태그 - 취소선

이것은 s 태그 입니다

u 태그 - 밑줄

이것은 u 태그 입니다
